Capstone Project - Kung-Fu Fight

DESCRIPTION

A simple martial arts fight game made with javaScript for capstone Project WD101. Made solely with the assets provided. No external Images used. The images of the Player is flipped horizontally using JS to create the Opponent.

POSSIBLE MOVES

  • d --> move forward
  • a --> move backward
  • Arrow Right --> Punch
  • Arrow Left --> Kick
  • Arrow Up --> Block

RULES

Reduce the opponent's health by kicking in the face or punching in the chest. Remember, punching in the chest will cause NO damage, if opponent blocks it. Opponent has slightly larger damage ration than Player. Opponent will always try to be in range to kick. Opponrnt will move back if a consequetive damage is taken.

Thats it..Enjoy...
